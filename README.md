# zopfliCI

CI repository that will build the latest zopfli [master branch](https://github.com/google/zopfli/tree/master) as a shared library for [all available platforms and architectures](https://docs.github.com/en/actions/using-github-hosted-runners/using-github-hosted-runners/about-github-hosted-runners#standard-github-hosted-runners-for-public-repositories) on Github

Supported OS:

- Windows
- macOS
- Linux

Supported architectures:

- x86 ( Windows only )
- x86_64
- arm ( Linux only )
- arm64
